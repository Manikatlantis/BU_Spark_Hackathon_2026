P.A.P.I — Point-cloud Analysis for Pathway Inclusion

P.A.P.I

Automated sidewalk hazard detection using high-density LiDAR data across Brookline, MA.

Raw LiDAR Geometry

Millions of 3D points (.laz) captured by Cyvl.ai represent real-world sidewalks in millimeter precision.

2 Meter Corridor Precision

Sidewalk centerlines are buffered in UTM space to isolate the exact walkable surface.

Plane Fitting

z = ax + by + c

Least-squares regression models surface tilt from the clipped point cloud.

Cross-Slope Calculation

|a| / √(1 + a² + b²)

ADA maximum cross-slope: ≤ 2%.

Vertical Step Detection

Δz > 6mm indicates cracks or uplift. Δz > 13mm is high severity.

Impact

250 corridors analyzed. 32,467 hazards identified.
